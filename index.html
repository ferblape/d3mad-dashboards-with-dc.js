## What is dc.js? > Dimensional Charting Javascript Library - Project page: http://dc-js.github.io/dc.js/ - Source code: https://github.com/dc-js/dc.js - Annotated source code of the example page: http://dc-js.github.io/dc.js/docs/stock.html
## What is dc.js? - On top of [D3.js](http://d3js.org/) and [Crossfilter](http://square.github.io/crossfilter/) - Create visualizations for filtering and aggregating data
## Our example dataset: Spanish municipalities budgets ![](images/dataset1.png)
## Super-quick introduction to crossfilter.js
## crossfilter.js - http://square.github.io/crossfilter/ - https://github.com/square/crossfilter/wiki/API-Reference
## crossfilter.js: concepts - Dimensions - Groups - (there is a limit of 32 concurrent dimensions)
### Dimensions - Dimensions are just the data concepts we are going to work with - Example: budget, population, budget per inhabitant
### Groups - Filters: the concepts we are going to filter by - Example: year, province... - Groups are the result of filtering the dimensions - Example: budget per year, mean budget per province
